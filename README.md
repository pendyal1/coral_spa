# Coral Spa Website

Static modern website for Coral Spa with Home, About Us, Services and Contact Us pages.

Run locally:

```sh
npm run dev
```

Then open `http://localhost:4173`.

## Deploy to GitHub Pages

This is a plain static site, so GitHub Pages can publish directly from the repository root.

1. Push changes to the `main` branch.
2. In GitHub, open the repository settings.
3. Go to `Pages`.
4. Under `Build and deployment`, set `Source` to `Deploy from a branch`.
5. Set `Branch` to `main` and folder to `/(root)`.
6. Click `Save`.
7. The site will deploy at `https://pendyal1.github.io/coral_spa/`.
